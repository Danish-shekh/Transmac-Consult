## Images & Media Guidelines

### Recommended Image Sizes:
- **Hero/Banner images:** 1920x600px
- **Team photos:** 800x800px (square)
- **Project photos:** 1200x800px (landscape)
- **Office photos:** 1200x800px (landscape)
- **Icons/logos:** SVG or PNG with transparency

### Image Formats:
- Preferred: JPEG (for photos), PNG (for logos/graphics)
- Optimize all images before uploading
- Maximum file size: 500KB per image

### Image Locations:
Place all images in: `wwwroot/images/`

Suggested folder structure:
```
wwwroot/images/
├── about/          (company photos)
├── team/           (founder & leadership photos)
├── services/       (service-related images)
├── projects/       (project photos)
└── offices/        (office photos)
```
